# 开发与扩展

## 目录约定

### 路由

- `src/router/static-routes.ts`：固定入口路由
- `src/router/module-routes.ts`：模块聚合
- `src/router/modules/*.ts`：Root.children 模块路由
- `src/router/dynamic-routes.ts`：菜单转动态路由

### 状态

- `src/stores/modules/auth.ts`
- `src/stores/modules/menu.ts`
- `src/stores/modules/permission.ts`

### 指令

- `src/directives/modules/*.ts`
- `src/directives/index.ts` 统一导出

## 新增一个后台页面

1. 新建视图文件到 `src/views/...`
2. 选择接入方式：
   - 固定路由：在 `src/router/modules/*.ts` 增加模块路由
   - 动态菜单路由：由后端/Mock 菜单返回对应 `component`
3. 如需按钮权限，添加 `v-permission` 并配置权限码

## 新增一个路由模块

在 `src/router/modules` 新建文件并导出 `routes` 或 `default`：

```ts
import type { RouteRecordRaw } from 'vue-router'

export const routes: RouteRecordRaw[] = [
  {
    path: 'report',
    name: 'ReportPage',
    component: () => import('@/views/dashboard/DashboardPage.vue'),
    meta: { title: '报表中心', requiresAuth: true }
  }
]
```
